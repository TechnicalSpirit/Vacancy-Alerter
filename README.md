Это приложение позволяет пользователю быть в курсе появления новых вакансий.

Информация о вакансиях сохраняется в Excel-файл и передаётся в Telegram через бота.
Уведомление содержит все вакансии, опубликованные за предыдущий день.

Dockerhub: https://hub.docker.com/r/technicalspirit/vacancy-alerter
